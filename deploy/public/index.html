🍳 CSV Recipe Import

📄 Single File Import

📝 Format: recipeName, description, preparation, ingredientName, quantity, unit

🔄 Multiple rows per recipe: One row for each ingredient

📄📄 Two File Import

📄 File 1 (Info): recipeName, description, preparation, cuisine, mainIngredient, season, image, ingredientsText, persons

📄 File 2 (Ingredients): recipeName, ingredientName, quantity, unit

🔗 Matching: Files linked by recipeName

📄 Recipe-Only Import

📄 Format: recipeName, description, preparation, cuisine, mainIngredient, season, image, ingredientsText, persons

🥘 Perfect for: Recipes with only text-based ingredient lists

⚡ Fast: No ingredient matching required

📦 Ingredient names: Must match existing products exactly

📏 Units: g, kg, ml, l, pcs, tsp, tbsp, cup, pinch, cl

💡 Two-file mode: Select info file first, then ingredients file

🍳 Multi-Device Workflow:

📱 iPhone: Perfect for shopping - quick access to lists in-store

💻 Mac & iPad: Ideal for menu planning, recipe management, and data entry


📊 How to Sync:

1. Export: Use 📤 Export on your current device to save all data

2. Transfer: Share the "grocery-data.json" file via iCloud, email, or AirDrop

3. Import: Use 📥 Import on your other device to load the data


💡 Tip: Export from your main device (Mac/iPad) regularly and import to iPhone before shopping
